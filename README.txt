nhentaidownload.py  此程式為n站番號下載器執行之後輸入番號即可
此程式需要套件 Beatifulsoup4
              requests
wnacgdownload.py    此程式為w站番號下載器執行之後輸入番號即可
此程式需要套件 Beatifulsoup4
              requests
              selenium
              webdriver-manager
檔案下載之後的位置為此程式同個資料夾下 檔案(是一個資料夾)名稱為番號名稱
程式是用Python去寫的，所以要安裝Python的環境，不會裝的自己Google
wnacgdownload.py這個程式要使用務必使用Chrome瀏覽器，用pip套件去裝
